# Relatório de Bugs

## 1 - Função "adicao" adicionando o primeiro número 2x

Localização: Linha 4 de Calculadora.py

Descrição: Ao fazer a soma dos números, a função "adicao" está fazendo a soma do primeiro número inserido pelo usuário 2(duas) vezes, ao invés de apenas 1(uma)

Contexto: Python 3.12.4 | Comando: python Calculadora.py

Passo a Passo:

- Comando: python Calculadora.py
- Comando: 1
- Comando: 1
- Comando: 2
- Resultado: 4.0
- Comando: n
- Fim de passo a passo

Consistência: Sempre acontece.

Resultado Esperado: 1 + 2 = 3
Resultado Obtido: 1 + 2 = 4
